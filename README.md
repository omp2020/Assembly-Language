# Assembly-Language
  Using NASM Assembler for x86_64 ALP

# To install NASM Assembler on Ubuntu
  ```
  sudo apt-get install nasm
  ```
# Commands to run NASM Program 
  To compile the code
  ```
  nasm -f elf64 file_name.asm
  ```
  To link the code
  ```
  ld -o file_name file_name.o
  ```
  To execute the code
  ```
  ./file_name.exe
  ```
